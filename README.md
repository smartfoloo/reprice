# Reprice

Repriceは、大手不動産ポータルサイトで中古マンションの適正価格を表示することができるChrome拡張機能です。

### 対応サイト

SuumoとYahoo!不動産に対応しています。

### インストール方法

1. このリポジトリを`.zip`形式でダウンロードし、展開します。
2. お使いのChromiumブラウザを開き、アドレスバーに`chrome://extensions`を入力します。(ChromeとEdge以外のブラウザでは、`ブラウザ名://extensions`を開きます)
3. 右上の「デベロッパーモード」をオンにし、左上の「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
4. 展開したフォルダを選択し、読み込みます。
5. SuumoやYahoo!不動産にアクセスし、中古マンションをお好みの条件で検索すると、物件の適正価格が見れるようになります。
