# Reprice

Repriceは、大手不動産ポータルサイトで中古物件の相場価格を表示することができるChrome拡張機能です。
**⭐️スターよろしくお願いします！⭐️**

> [!NOTE]
> 完成次第Chromeウェブストアにアップロードする予定です。

## 対応状況

## インストール方法

1. このリポジトリを`.zip`形式でダウンロードし、展開します。
2. お使いのChromiumブラウザを開き、アドレスバーに`chrome://extensions`を入力します。(ChromeとEdge以外のブラウザでは、`ブラウザ名://extensions`を開きます)
3. 右上の「デベロッパーモード」をオンにし、左上の「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
4. 展開したフォルダを選択し、読み込みます。
5. 不動産ポータルサイトにアクセスし、中古マンションをお好みの条件で検索すると、物件の相場価格が見れるようになります。

## 相場価格の計算方法

### 中古マンション

1. 対象の物件と同じエリアかつ同じ築年数帯（10年ごと）での成約事例をもとに、似た物件の平均坪単価を計算します。
2. 対象の物件に坪単価を適用します。

### 中古一戸建て

1. 対象の物件と同じエリアかつ同じ築年数帯（10年ごと）での成約事例をもとに、似た物件のの坪単価をそれぞれ計算します。
2. 坪単価が中央値より20%以上高いまたは低い物件を除いた平均坪単価を計算します。
3. 対象の物件に坪単価を適用します。

> [!NOTE]
> 人口密度が1平方キロメートルあたり5000人以上の自治体の物件は「同じ最寄駅」が「同じエリア」の条件となりますが、それ以外の自治体の物件は「同じ町名」が「同じエリア」の条件となります。

## よくある質問

#### どういった物件に対応していますか？

中古マンションと中古一戸建てのみです。

#### どのページを開けば適正価格を見られますか？

SUUMO、アットホーム、Yahoo!不動産などで「中古マンション」または「中古一戸建て」を選択し、お好みの条件で検索すると見ることができます。個別の物件ページには対応していません。

#### この拡張機能をアップデートするためにはどうしたらいいですか？

現在公開中のものはベータ版ですので、アップデートにはご自身で随時ダウンロードし、再アップロードしていただくことが必要です。なお、このリポジトリをクローンすると、`git pull`のワンコマンドでアップデートすることができます。将来的にはChromeウェブストアにアップロードし、自動アップデートが可能になる予定です。

## 注意事項

- 相場価格はあくまで周辺相場をもとにした参考値であり、個別物件の価値を保証・評価するものではありません。
- 戸建ては土地と建物の評価が複雑に影響するため、マンションと比べて相場価格の精度が低くなる場合があります。
- 取引件数が少ない地域ではデータが限られるため、相場価格の精度が低くなることがあります（一般に都心部は取引が多く、精度が高くなります）。
