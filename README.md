# Reprice

Repriceは、大手不動産ポータルサイトで中古マンションの適正価格を表示することができるChrome拡張機能です。**⭐️スターよろしくお願いします！⭐️**

> [!NOTE]
> 完成次第Chromeウェブストアにアップロードする予定です。

## 対応サイト

SUUMO、アットホーム、Yahoo!不動産に対応しています。

## インストール方法

1. このリポジトリを`.zip`形式でダウンロードし、展開します。
2. お使いのChromiumブラウザを開き、アドレスバーに`chrome://extensions`を入力します。(ChromeとEdge以外のブラウザでは、`ブラウザ名://extensions`を開きます)
3. 右上の「デベロッパーモード」をオンにし、左上の「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
4. 展開したフォルダを選択し、読み込みます。
5. 不動産ポータルサイトにアクセスし、中古マンションをお好みの条件で検索すると、物件の適正価格が見れるようになります。

## 適正価格の計算方法

同じ駅・同じ築年数帯（1-10年, 11-20年など）の過去取引データから平均坪単価を計算し、その物件の面積に適用して適正価格を算出します。現在の取引データは2024年第4四半期から2025年第3四半期のものになっています。

## よくある質問

#### どういった物件に対応していますか？

中古マンションと中古一戸建てのみです。

#### どのページを開けば適正価格を見られますか？

SUUMO、アットホーム、Yahoo!不動産などで「中古マンション」または「中古一戸建て」を選択し、お好みの条件で検索すると見ることができます。個別の物件ページには対応していません。

#### この拡張機能をアップデートするためにはどうしたらいいですか？

現在ベータ版を公開しているだけなので、アップデートにはご自身で随時ダウンロードし、再アップロードしていただくことが必要です。なお、このリポジトリをクローンすると、`git pull`のワンコマンドでアップデートすることができます。将来的にはChromeウェブストアにアップロードし、自動アップデートが可能になる予定です。
